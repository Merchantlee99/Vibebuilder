# Memory Curation

Self-improvement is audited improvement of workflows, not autonomous self-modification.

## Required Evidence

- source event id or evidence id
- repeated validated occurrence count
- confidence
- expiry or review date
- proposed action
- test fixture
- reviewer verdict

## Output Contract

Return quarantine, reject, or promote-with-review. Raw learning logs never become active rules.
