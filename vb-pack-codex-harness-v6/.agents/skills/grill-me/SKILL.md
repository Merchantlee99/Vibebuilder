# Grill Me

Use this skill for early idea discovery or unclear user intent.

## Steps

1. Ask focused questions about user goal, constraints, risks, success criteria, and unwanted behavior.
2. Walk down dependency branches one decision at a time.
3. Stop when the remaining ambiguity is either resolved or explicitly recorded as a decision gap.
4. Convert useful answers into an intent route, PM brief, domain language candidate, or spec candidate.

## Output Contract

Return resolved decisions, unresolved questions, and the next artifact to update. Conversation is not authority until captured in spec, domain language, ADR, or review records.
