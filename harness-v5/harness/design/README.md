# Harness Design State

Mutable design artifacts belong here when a project needs durable UI/UX state.

Suggested files in real projects:

- `design-system/MASTER.md`
- `design-system/pages/<page>.md`
- `ui-audits/*.md`

For normal UI work, prefer project-facing artifacts under `docs/ai/current/` first:

- `UI-UX-Brief.md`
- `Design-System.md`
- `UI-Review.md`
